# KLAI 매력도 — 디자인 가이드

> 랜딩페이지 디자인을 클로드 디자인/아티팩트에서 수정하기 위한 디자인 시스템 정리.
> 실제 코드 토큰은 `app/globals.css`(`:root` 다크 + `.theme-light` 라이트). 랜딩은 **라이트 테마**를 사용합니다.
> 비주얼 목업은 `landing-design.html`(자체 완결 · 편집용).

---

## 0. 디자인 철학

- **ZeroSite × careet 라임 룩** — 어두운 네이비 기반의 데이터 톤 + careet식 콘텐츠-피드(매거진) 레이아웃 + 라임(샤르트뢰즈) 포인트.
- **라임은 "채움 전용"** — 큰 면적의 라임 텍스트는 가독성이 낮음. 그래서:
  - **채움(버튼·배지·핀)** = `--amber`(lime-500 `#84cc16`) + 다크 텍스트(`--on-accent`).
  - **텍스트 강조(링크·아이브로·태그)** = `--blue-l`(lime-700 `#4d7c0f`, 흰 배경 위 가독).
- **콘텐츠-피드 우선** — 거대한 마케팅 히어로 대신, 실데이터 카드(동네)가 먼저. careet처럼 카테고리 라벨 + 에디토리얼 헤드라인 + 상태 배지 + 북마크.
- **타이포 모더레이트** — 카드 타이틀 17px, 섹션 헤더 24~27px. 헤드라인·타이틀만 800. 과한 디스플레이 금지.
- **데이터 윤리** — 공개 등급은 거칠게(처방 중심), `샘플/잠정(Provisional)` 배지 필수. 가짜 데이터 금지.

| `--amber-d` | `#65a30d` | hover(lime-600) |

**등급 스케일 (KLAI 발산):** S `#0f6e5c` · A `#1e7a8c` · B `#3e9aa8` · C `#e2a33a` · D `#d2691e` · E `#a23a2a`
(녹청 S→A→B 에서 앰버 C → 오렌지 D → 적벽돌 E 로 발산. 낙인 방지 위해 거친 6단계.)

**3D 지도 색:** 상승 = 초록(`#16a34a`/`#34d399`), 하락 = 로즈(`#f43f5e`/`#fb7185`). 다크 패널 = `#0D2B5E`.

> 다크 테마(`:root`)는 라임 형광(`--blue-l #b6f24e`, `--amber #a3e635`) + 네이비 배경(`#0a0a0b`). /map 등 다크 화면용.

## 2. 타이포그래피

- **본문**: Pretendard (`--font`). 400~600.
- **디스플레이/숫자**: Poppins (`--font-display`) — 아이브로, 큰 숫자(KLAI 점수)는 `tabular-nums`.
- 스케일:
  - 디스플레이/히어로: `clamp(27px, 4vw, 34px)` · 800 · letter-spacing −0.02em
  - 섹션 헤더 `h2`: 24~27px · 800
  - 카드 타이틀 `h3`: 17px (피처드 큰 카드 24px) · 800
  - 본문: 13~15px · 400~600 · `--muted`
  - 아이브로(`klai-eyebrow`): 12px · 800 · **대문자 · tracking 0.28em** · `--blue-l`
  - 메타/캡션: 11~12px · `--muted2`

## 3. 간격·레이아웃

- **컨테이너**: `max-width: 1152px`(max-w-6xl), 좌우 패딩 24px.
- **풀폭 히어로**: 3D 라이브맵은 컨테이너 밖 100% 폭(72vh, min 540px).
- **섹션 패딩**: 32~48px 상하.
- **그리드**: 카드 피드 `repeat(3, 1fr)` (모바일 1열), gap 16px. 피처드 `1.55fr / 1fr`.
- **라운드**: 카드 16px, 큰 패널/CTA 18~28px, 칩/배지 999px(pill).
- **그림자**: 평소 거의 없음(보더로 구분). hover 시 `lift`로 −4px + 라임 링.

## 4. 핵심 컴포넌트

### 버튼
- **채움(CTA)**: `bg --amber` + `--on-accent` 텍스트 + `btn-glow`(hover 시 라임 글로우). 예: "지번 진단", "무료 구독".
- **고스트**: `bg --card2` + `--line` 보더 + `--ink`. hover 시 라임 보더.

### 배지·태그
- **상태 배지(`status-pill`)**: pill, 800, 11.5px. 유행중=`--amber`/onaccent, 식는중=다크/흰글씨.
- **카테고리 칩**: 로컬 동네 카테고리 — 핫플·젠트리(amber) / 로컬크리에이터(green `#16a34a`) / 예술·메이커(blue `#1E5FA8`) / 관광·라이프(teal `#0F6E5C`).
- **klai-tag**: 라임 rgba 배경 + lime-700 텍스트 — "에디토리얼 큐레이션" 등.
- **cat-tag**: 11px · 800 · tracking 0.06em · `--blue-l` — 카드 위 시군구·업종.
- **hl-mark**: careet 시그니처 형광펜 — `accent-bright` 65% 언더라인 그라데이션.

### 기사형 카드(ArticleCard)
- **커버**(aspect 16:9): 실제는 **CARTO voyager 지도 타일**(동 중심 2×2, 정중앙). 위에 등급 그라데이션 스크림(하단만 어둡게) + **유행중/식는중 배지**(좌상) + **북마크**(우상) + **이유 콜아웃 핀**(중앙, 흰 라벨+점) + **KLAI 큰 숫자**(좌하, tabular).
  - *목업에선 지도풍 그라데이션으로 대체. 디자인 수정 시 커버 비율·스크림·핀 위치 유지.*
- **본문**: 카테고리(cat-tag) → 에디토리얼 헤드라인(데이터로 생성: "검색·미디어가 먼저 움직였다", "거래절벽 — 위기 경보" 등) → (피처드만 발췌+칩) → 메타("동명 · 이번 주 · Flagtale Weekly").

### 3D 라이브맵 히어로
- 풀폭 다크맵 + 3D 컬럼(기둥=KLAI, 초록 상승/로즈 하락). 오버레이: 타이틀(좌상) · 컨트롤 pill(우상: 전체/상승/하락/라벨/자동회전) · **동네 상세 패널**(우측) · 범례(좌하).
- **상세 패널**: 등급 배지 + KLAI(32px) + 모멘텀 + **4축 레이더**(또래 평균 점선 겹침) + 칩(젠트리/시장/이유) + CTA.

### 4축 레이더(MiniRadar)
- 인구(상)·상권(우)·공간(하)·인식(좌). 동심원 4링 + 라임 채움 폴리곤 + **또래(같은 지역) 평균 = 점선**.
- 다크 패널: grid `rgba(255,255,255,.14)`, 라임 `#9be15d`. 라이트 모달: grid `rgba(15,30,60,.12)`, 라임 `#65a30d`.

### 데이터 마퀴
- 무한 가로 흐름(30s linear). 양끝 페이드 마스크. 출처 칩 2벌 반복.

## 5. 랜딩 섹션 순서

1. **3D 라이브맵 히어로**(풀폭) — "지금 움직이는 동네만, 지도 위 3D로"
2. **요약 스탯 + 모멘텀 막대그래프**
3. **인트로 + 매거진 피처드**(큰 카드 1 + 사이드 2) — "지금 전국에서 뜨는 동네를 읽다"
4. **🔥 지금 뜨는 동네**(캐러셀)
5. **🗂 전국 동네 피드**(탭: 전체/뜨는/식는/젠트리)
6. **📰 지금 뜨는 로컬 동네**(네이버 실시간 + 클릭 멀티플랫폼 상세, 검색 급등順)
7. **데이터 소스 마퀴**
8. **🗺/🔎/🏪/🏛 활용 4칩**
9. **뉴스레터 CTA** — "트렌드를 먼저 읽으세요"
10. **푸터**

## 6. 수정 시 체크리스트

- [ ] 등급 색은 6단계 발산 스케일 그대로(낙인·시세표화 방지).

- [ ] 타이포 모더레이트(카드 17 / 헤더 24~27, 과한 디스플레이 금지).
